# nim-edlib
Nim wrapper for [edlib](https://github.com/Martinsos/edlib) (Martin Šošić)

## Requirements

* nim compiler
  * c compiler (e.g. clang or gcc)
* `pkg-config`
* "edlib" installed
  * We use `pkg-config --cflags edlib`

## Versions

* `edlib-1.2.6` is the first version we ran **c2nim** on.
